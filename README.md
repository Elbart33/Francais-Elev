# Francais Elevate — PWA

Application mobile pour développer son lexique Francais marocain niveau C1/C2.

## ⚡ Installer sur iPhone (12 minutes)

### Étape 1 — Activer GitHub Pages

1. Va sur **github.com** → crée un compte si besoin
1. Clique **“New repository”**
1. Nom : `Francais-elevate` · Coche **“Public”** · Clique **“Create repository”**
1. Upload tous les fichiers de ce dossier (drag & drop sur la page du repo)
1. Va dans **Settings → Pages → Source** → choisir **“main”** → Save
1. Ton URL sera : `https://TON-USERNAME.github.io/Francais-elevate`

### Étape 2 — Installer sur iPhone

1. Ouvre **Safari** (obligatoire, pas Chrome)
1. Va sur ton URL GitHub Pages
1. Appuie sur **⬆ Partager** (en bas au centre)
1. Scroll et tape **“Sur l’écran d’accueil”**
1. Tape **“Ajouter”** en haut à droite

✅ L’app apparaît sur ton écran d’accueil et fonctionne **100% hors ligne**.

## Fonctionnalités

- 15 mots C1/C2 avec nuances, synonymes, exemples naturels
- SRS (spaced repetition) — mémorise ta progression
- Mode Immersion — textes en Francais avec mots cliquables
- Quiz automatique
- Suivi de progression par niveau et thème
- Streak quotidien
- Offline complet via Service Worker
